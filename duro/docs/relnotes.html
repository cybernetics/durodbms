Release Notes for Duro 0.9

Nested transactions

The execution of a parent transaction is suspended while a child transaction executes. It is an error to perform an operation under the control of a transaction which has a running child transaction.

Creating libraries

Libraries for user-defined types and operators must be created using GNU Libtool. The name of the library is passed without the file extension.

Performance

Duro 0.9 supports hash and B-tree indexes, and does some amount of query optimization.

For each candidate key, a hash index is created automatically.

Expressions

Duro 0.9 expressions may not contain relation-valued expressions which contain variables referring to variables in the enclosing expression. In other words, a relation-valued RDB_expression may only represent a table, but not a general relation-valued expression.
