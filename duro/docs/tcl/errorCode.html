Durotcl - errorCode

If a Durotcl command fails for a Duro error, the global Tcl variable errorCode is set to a list of the format {DURO error}.

If a transaction was available when the error occured, error is the error value as a Tutorial D literal. Otherwise, it is the name of the error type. See the error types section in the Duro API specification for a list of system-provided error types.
